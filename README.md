# AI Medical Device After-Sales Workflow Prototype

This project demonstrates how Generative AI can assist medical device
technical support teams by automating issue analysis and troubleshooting guidance.

## Workflow

Device Issue Input  
↓  
AI Diagnosis  
↓  
Structured JSON Output  
↓  
Excel Service Record  
↓  
PDF Troubleshooting Report  
↓  
REST API Integration

## Features

- Prompt-based troubleshooting workflow
- Automated report generation
- Excel service log database
- FastAPI REST interface
- JSON structured output
